# Avoid-Eagle
- 개발인원 : 1명
- 역할
  - 전체
  
## Introduction
무작위로 날아오는 독수리들을 피해 하늘 높이 올라가야 하는 간단한 피하기 게임입니다. 

사용하면 화면상의 모든 독수리가 사라지는 폭탄이라는 특수 아이템을 구현하여 운적인 요소를 최소화 했습니다. 

올라간 높이로 순위가 매겨져 다른 유저와 경쟁할 수 있습니다.
